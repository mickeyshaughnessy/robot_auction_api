Robot Auction API Documentation

Authentication Endpoints

The RSX alpha API is currently running at 100.26.236.1:5001

POST /register

Register a new user account.

Request Body:

ParameterTypeDescription
usernamestringDesired username
passwordstringDesired password

Response:

Status 201: User registered successfully

POST /login

Authenticate a user and receive an access token.

Request Body:

ParameterTypeDescription
usernamestringUser's username
passwordstringUser's password

Response:

Status 200: Login successful, returns access token
Status 401: Invalid username or password

Account Management

GET /account_data

Retrieve the account data for a user.

Query Parameters:

ParameterTypeDescription
usernamestringUser's username
stringUser's password

Response:

Status 200: Returns account data
Status 400: Username is required
Status 404: Account not found

Buyer Endpoints

POST /make_bid

Submit a new bid for a robot service.

Request Body:

ParameterTypeDescription
servicestringType of service requested
latfloatLatitude of service location
lonfloatLongitude of service location
pricefloatBid price
end_timeintUnix timestamp of bid expiration

Response:

Status 200: Bid submitted successfully, returns bid ID
Status 403: Insufficient funds or other error

POST /nearby

Retrieve nearby robot service activities.

Request Body:

ParameterTypeDescription
latfloatLatitude of search center
lonfloatLongitude of search center

Response:

Status 200: Returns list of nearby activities

Seller Endpoints

POST /grab_job

Attempt to accept an available job for a robot.

Request Body:

ParameterTypeDescription
servicesarrayList of services the robot can perform
latfloatCurrent latitude of the robot
lonfloatCurrent longitude of the robot
max_distancefloatMaximum travel distance for the robot

Response:

Status 200: Job successfully grabbed, returns job details
Status 204: No matching jobs available

System Check

GET/POST /ping

Check if the API is operational.

Response:

Status 200: API is operational
